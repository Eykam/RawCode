Relational Model and SQL -- also, brief discussion of Web Application Development

Here we go deeper into the relational model and cover SQL, as well as relational algebra and formal semantics underlying SQL. We also go through how database-backed Web Applications are architected, and discuss Python Django framework in some detail (Python Django is not covered in the Fall 2021 edition).
